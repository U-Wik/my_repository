Calculates a first order polynom for a given x as a0 + a1*x

Parameters
afirst number, integer or float
bfirst number, integer or float
Returns
float

Description of an anonymous value

Examples

simple_math.poly_second(x, a0, a1, a2)¶

Calculates a second order polynom for a given x using poly_first as poly_first(x, a0, a1) + a2*(x**2)

Parameters
afirst number, integer or float
bfirst number, integer or float

float

Description of an anonymous value

Examples
